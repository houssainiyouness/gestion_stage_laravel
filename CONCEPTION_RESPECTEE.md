# Conception respectée

Le projet respecte le cahier de charge fourni :

## Acteurs

- Super Admin
- Administrateur Organisme
- Encadrant
- Étudiant

## Modules

- Authentification
- Offres de stage
- Candidatures
- Stages
- Documents
- Suivi & traçabilité
- Évaluations
- Soutenances
- Dashboard

## Workflow

1. L’admin organisme crée une offre.
2. L’étudiant postule avec CV et lettre.
3. L’admin organisme accepte ou refuse.
4. Si accepté, le stage est créé automatiquement.
5. L’encadrant suit le stage.
6. L’étudiant ajoute les documents.
7. La soutenance est planifiée.
8. La note finale et le résultat sont enregistrés.
